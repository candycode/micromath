Detailed Description

Memory tracer. Modified version of code found on www.relisoft.com Usage: store each pointer into map when memory allocated; remove pointer from map when memory deallocated; at destruction time it checks if map size is zero i.e. number of allocations == number of deallocations in the destructor. In case map size not zero it prints a list of lines where non-deallocated memory was allocated.
