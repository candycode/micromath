Detailed Description

template<class ValT>
struct mmath_plus::rte< ValT >

Run-time environment. Used to store:

  • functions
  • variables
  • constants
  • program (could be stored outside the RTE)
  • value std::stack
  • execution std::stack
  • instruction pointer
